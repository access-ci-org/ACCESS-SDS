Use Case: Compression and decompression of files.

Code details and examples:

1. Compression:

Code:

```bash
bzip2 filename
```

This command will replace filename with filename.bz2 in compressed form.

Input File: filename (any file)

Format Required: The original file could be in any format (txt, data, etc.)

Specifications and Details: None. The original file can contain any data.

2. Decompression:

Code:

```bash
bunzip2 filename.bz2
```

This command will extract the original file from filename.bz2.

Input File: filename.bz2 (BZ2 file)

Format Required: .bz2 (Bzip2 Compressed File)

Specifications and Details: The compressed file should be a valid BZ2 file.
